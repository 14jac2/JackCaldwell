Rent or Buy?

The following tool will evaluate whether it is worth purchasing a home, provided a set of mortgage parameters and reference rental rate. In addition, the tool indicates the amount the home must appreciate for it to be worth purchasing, if applicable.

The model currently considers:

  • Mortgage insurance
  • Land transfer tax
  • Interest paid on loan
  • Home maintenance fees
  • Property taxes paid
  • Investment returns were the user to invest the downpayment instead
  • Rental rate increase year on year

And makes the following assumptions:

  • Investment gains equal to mortgage rate
  • All the rates to be posted soon...
Here is the equations for buy and rent sunk costs, respectively: \( C_b = I_{mortgage} + T_{prop} + M + L + T_{land}\), \( C_r = R - I_{inv}\)
